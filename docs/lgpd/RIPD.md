# RIPD - Relatório de Impacto à Proteção de Dados

Versão: 2026-04-24

## Escopo inicial

- Tratamento de dados pessoais de titulares da plataforma.
- Tratamento de dados operacionais de clientes das barbearias.
- Tratamento de dados de equipe (incluindo data de nascimento opcional).

## Metodologia

- Inventariar operações (ROPA).
- Classificar risco (probabilidade x impacto).
- Definir controles existentes.
- Definir plano de mitigação e risco residual.

## Itens pendentes

- Matriz de risco por operação.
- Plano formal de resposta a incidente de dados pessoais.
- Critérios de retenção e eliminação por categoria de dado.

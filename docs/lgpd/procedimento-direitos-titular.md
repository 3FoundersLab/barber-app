# Procedimento de Atendimento aos Direitos do Titular (Art. 18)

Versão: 2026-04-24

## Canais

- API interna: `POST /api/lgpd/solicitacoes`
- E-mail: privacidade@barbertool.com.br

## Fluxo mínimo

1. Receber e registrar solicitação com protocolo.
2. Validar identidade do titular.
3. Classificar tipo de direito solicitado.
4. Executar ações técnicas necessárias.
5. Responder no prazo legal aplicável.
6. Manter trilha de auditoria do atendimento.

## Tipos contemplados (inicial)

- Confirmação de tratamento.
- Acesso.
- Correção.
- Anonimização, bloqueio ou eliminação.
- Portabilidade.
- Informação de compartilhamento.
- Revogação de consentimento.
- Revisão de decisão automatizada (quando aplicável).

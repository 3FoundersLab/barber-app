# Política de Retenção e Eliminação de Dados

Versão: 2026-04-24

## Diretriz

Dados pessoais devem ser mantidos somente pelo prazo necessário para cumprimento de finalidades legítimas, obrigações legais/regulatórias e defesa de direitos.

## Regras iniciais

- Dados de sessão: conforme expiração técnica de autenticação.
- Consentimentos: manter histórico para prova de conformidade.
- Solicitações LGPD: manter protocolo e trilha de atendimento.
- Logs operacionais: revisar e aplicar minimização/anonimização periódica.

## Ações obrigatórias

- Definir prazos por tabela com base legal.
- Implementar rotina automática de descarte/anonimização.
- Revisar periodicamente os prazos.
